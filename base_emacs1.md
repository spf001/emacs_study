# Emacs的基本组合键 #
##光标移动##
* C-h C-a:显示帮助
* C-x C-c:退出emacs
* C-x C-s:保存当前Buffer
* C-x C-f:打开和新建文件
* C-x C-w:使用其他文件名另存为文件
* C-x C-v 关闭当前缓冲区文件并打开新文件
* C-n 光标移动到下一行
* C-p 光标移动到上一行
* C-k 删除一行(光标在行首)
* M-x 执行命令
* M-v 向前滚动一个屏幕
* C-v 向后滚动一个屏幕
* C-a 移动到行首
* C-e 移动到行尾
* M-a 移动到句首
* M-e 移动到句尾
* C-l 重新绘制屏幕，并将光标所在行至于屏幕中央
* M-< 移动到文件开始
* M-> 移动到文件结束
* C-x 1 关掉其他所有窗口，只保留一个窗口
* C-@ C-w移出一段文字
* C-@ M-w复制一段文字
* C-y粘贴一段文字到光标指定地方
* C-y召回最近移除的文字
* M-y召回移除的文字

##查找##
* 文件内查找C-s(支持正则表达式)
* 文件内反向查找C-r(支持正则表达式)
* 文件内替换 M-x replace-string RET

##杂项##
* 设置默认工作目录 (setq default-directory "your directory")
* 缓冲区切换 C-x b
* 缓冲区关闭 C-x k
* 跳转到制定行 M-x goto-line
